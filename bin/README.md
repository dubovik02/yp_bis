# bis_rust - CLI приложение для чтения данных о транзакциях из двух файлов и сравнения.

---

## Технологии

Проект создан на базе:

-   Rust, Cargo

## Реализованный функционал

CLI приложение для чтения данных о транзакциях из двух файлов и сравнения.

## Запуск проекта

cargo run --bin comparer -- --file1 <filename> --format1 txt|csv|bin --format2 txt|csv|bin --file2 <filename>
